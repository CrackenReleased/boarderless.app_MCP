📖 Security & Consent
Why does it require remote debugging?
The bridge securely attaches to a standard browser tab. This is how the AI reads the canvas spatial ledger and makes layout modifications.
Do I have to approve actions?
Yes! A glassmorphic permission drawer rises on the canvas page before any mutation, giving you absolute control to accept or reject edits.

Select Browser Profile Mode

To connect the canvas, we need to open a browser window with remote debugging enabled. Select your profile option below:

👤 Personal Profile (Recommended)

Full Sync

Loads your normal Chrome/Edge browser. You'll keep your active logins, history, and Google Drive sync.

⚠️ Close Chrome First: You must save your work and completely close all active Chrome/Edge windows so the debugger can launch with your profile.

🛡️ Isolated Profile

Sandbox

Launches a clean temporary window with a separate profile. Fast and safe, but requires logging in again to save to Google Drive.
